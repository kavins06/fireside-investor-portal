For the General Partner

Add a deal just by talking to Claude

No website logins, no spreadsheets, no code. Set it up once — about a minute — and from then on, putting a deal on the site is a normal conversation.

Two things you'll need (sent with this guide)
A connector link
A web address
A publish code
Your private password — keep it to yourself
1

Connect Fireside to Claude

In Claude, open Settings → Connectors, click Add custom connector, and paste this link — https://portal-eta-peach.vercel.app/api/mcp — then leave everything else blank and click Add. If it warns the connector "isn't verified," that's normal for an in-house tool — continue.

Then in a chat, switch on “Fireside Publish” using the + button next to the message box.
2

Tell Claude about the deal

Start a chat and say: “Add a new deal to the Fireside website.” Then paste the details or attach the pro forma / offering memo.

The first time, Claude asks for your publish code — paste it and say “remember this,” so you won’t be asked again.
3

Review & go live

Claude builds the deal, fills any gaps with Fireside’s standard assumptions (and tells you which), then shows you the projected returns. You confirm — and it’s on the website in about a minute.

You can’t publish broken numbers. Claude checks every deal against the real returns model first — if something’s off, it tells you in plain English and fixes it before anything goes live.

You can also just say…

  • “What deals are live right now?”
  • “Update the Maple St deal — exit cap 6.5%.”
  • “Keep this one off the homepage — or remove it.”

If something’s off

  • “Incorrect publish token” → re-paste your code.
  • Don’t see Fireside Publish? → switch it on with the + button.
  • Claude lists problems? → just answer its questions.
